# 🧠 AI Text Analysis App

Streamlit app powered by Azure AI Language that analyzes raw text and detects entities, PII, and language automatically.

## 🚀 Features
- 🌍 Language detection  
- 🏷️ Named entity recognition  
- 🔒 PII detection (emails, phone numbers, etc.)  
- 🧹 Redacted output text  

## 🛠 Tech Stack
- Streamlit  
- Azure AI Language  
- Python  
- Azure Cognitive Services  

## 💡 Use Cases
- Document redaction  
- Resume parsing  
- Compliance scanning  
- Customer support automation  
- Data extraction
