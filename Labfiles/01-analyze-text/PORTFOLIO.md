# Max Magulick — AI Automation Portfolio

## 🧠 AI Text Analysis App

AI-powered text analysis app built using Azure AI Language and Streamlit.

### What it does

This app analyzes raw text and automatically:

* Detects language
* Extracts named entities
* Identifies PII (emails, phone numbers)
* Generates redacted text output

### Tech Stack

* Python
* Streamlit
* Azure AI Language
* Azure Cognitive Services
